## 🤝 Contributing / 贡献

Contributions are welcome! Please feel free to submit a Pull Request. This repo is auto-updated every 3 days, but manual additions are always appreciated.

欢迎贡献！请随时提交 Pull Request。本仓库每 3 天自动更新，但也欢迎手动添加。

### How to add a new awesome repo / 如何添加新的 awesome 仓库

1. Fork this repository
2. Edit `data/repos.yml` to add your entry
3. Submit a Pull Request

### Auto-discovery / 自动发现

This repository uses GitHub Actions to automatically discover new awesome-agent repos. The CI workflow:

1. Searches GitHub for new awesome-agent repositories
2. Fetches star counts and descriptions via GitHub API
3. Categorizes repos using keyword matching
4. Updates README.md and data files
5. Commits changes if any updates are found
